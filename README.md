SECUREXFER v3.0 🔒

SecureXfer v3.0 is a secure file transfer and monitoring system developed using Python and Streamlit. The project provides encrypted file transfer, SHA-256 integrity verification, alert monitoring, audit logging, and report generation through an interactive web dashboard.

🚀 Features
Secure file transfer with encryption
SHA-256 file integrity verification
User authentication system
Sensitive file classification
Real-time alert monitoring
Audit logging with SQLite
TXT, CSV, and JSON report generation
Interactive dashboard with visual analytics
Role-based user access

🛠 Technologies Used
Python
Streamlit
SQLite
Cryptography Library
Pandas
Plotly
Hashlib

⚙️ System Workflow
Monitor file transfer activities
Classify sensitive files
Generate SHA-256 hashes
Authorize file operations
Trigger alerts for suspicious activities
Generate audit reports
